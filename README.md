# nodigram

Instagram clone with Express + Prisma + React and React Native

## User Stories

- [x] Create Account
- [x] Request Secret
- [x] Confirm Secret (Login)
- [ ] Like/Unlike a photo
- [ ] Comment on a photo
- [ ] Search by User
- [ ] Search by location
- [ ] Follow/Unfollow User
- [ ] See user profile
- [ ] See the full photo
- [ ] Edit my profile
- [ ] Upload photos
- [ ] Edit/Delete the photo
- [ ] See the feed
