# nodigram

Instagram clone with Express + Prisma + React and React Native

## User Stories

- [x] Create Account
- [x] Request Secret
- [x] Confirm Secret (Login)
- [x] Like/Unlike a photo
- [x] Comment on a photo
- [x] Search by User
- [x] Search Post
- [x] Follow/Unfollow User
- [ ] See user profile
- [ ] See the full photo
- [ ] Edit my profile
- [ ] Upload photos
- [ ] Edit/Delete the photo
- [ ] See the feed
